Vibe Island turns the MacBook notch into a real-time control surface for AI coding agents. See what every session is doing at a glance, approve permission requests without switching windows, answer agent questions with one click, and jump straight back to the exact terminal tab or IDE window that needs you.

## Install

Download the latest release from **[vibeisland.app](https://vibeisland.app)**, or install with Homebrew:

```
brew install --cask vibe-island
```

---

## Why Vibe Island

- **Live session status** — every running agent visible in the notch: working, waiting for input, needs approval, done
- **Approve from the notch** — permission requests pop up as cards; allow or deny without hunting for the right terminal
- **Answer questions in one click** — when an agent asks you to choose, pick the option right in the panel
- **Precise jump-back** — click a notification and land on the exact terminal tab, split pane, or IDE window that asked
- **Plan review** — read full Markdown plans and send feedback before an agent starts editing
- **Usage limits** — see subscription quota windows per provider before you hit the wall
- **SSH Remote** — monitor agents running on remote servers from your Mac's notch
- **Zero config** — detected CLIs are set up automatically on first launch
- **Native Swift** — under 50 MB RAM, near-zero idle CPU, no Electron; everything stays on your machine
